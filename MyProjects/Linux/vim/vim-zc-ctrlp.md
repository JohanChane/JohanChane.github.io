### 使用

`C-p` 进入 ctrlp 的 file mode 模式。输出 `?` 查看帮助。


- 搜索模式
    files, buffers, mru files(搜索 Files, Buffers and MRU files)
    C-f, C-b

- 搜索的路径是 PWD

- 根据文件名搜索还是路径搜索

    C-d

- 是否开启 regex

    C-r

- 历史命令
    
    C-p, C-n

- 浏览结果
    
    C-j, C-k, <Up>, <Down>

- 打开多个文件
    
    C-z + C-o

- 返回上层目录
    `..` 会修改 PWD

*`help ctrlp-mappings`, F5 是刷新*

*不支持中文输入。。。*
